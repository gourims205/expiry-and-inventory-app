# Expiry and Inventory Management App
## Overview
A Flutter-based mobile application for managing inventory and tracking product expiry dates through automated notifications and barcode scanning.

## Features
- Inventory Management
- Expiry Date Tracking
- Barcode Scanning
- Automated Notifications
- Firebase Authentication
- Cloud Firestore Integration

## Technologies Used
- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
